# KATONG IP

## 角色定义

KATONG 是本 skill 中文正文配图的固定视觉 IP。原型是「亚洲男性创业者」的极简扁平矢量形象。

默认每张图都要出现 KATONG。KATONG 不是吉祥物，不是贴纸，不是可爱装饰，而是正在认真参与系统运转的创业者/操作者，承担画面的核心动作。

## 外形（每张图必须保持一致）

- 亚洲男性创业者形象，极简扁平矢量风格（minimalist flat vector）。
- 短黑色寸头（short black buzz cut）。
- 浓眉，细长眼（narrow eyes），清晰下颌线（defined jawline）。
- 表情冷静、自信、专注（calm confident expression），不夸张。
- 黑色连帽衫（black hoodie）。
- 皇家蓝裤子，色值约 #4A78F0（royal blue pants）。
- 胸前有干净的白底徽章，上面写着数字 34（white chest badge with number 34）。
- 几何矢量插画、干净轮廓线（geometric vector illustration, clean outlines）。
- 姿态根据配图需要动态调整（dynamic pose），但上述五官 / 服装 / 徽章特征在每张图必须保持一致。

## 风格禁忌（防止画风漂移）

- 不要 anime / manga（日漫风）。
- 不要 Pixar / 3D render（3D 渲染或皮克斯风）。
- 不要 photorealistic（写实照片风）。
- 不要过度可爱、幼稚、Q 版。
- 不要给角色加复杂配饰、表情包、闪亮大眼。
- 不要让 KATONG 只是站在角落里当装饰。

## 性格

- 冷静、自信、专注的创业者气质。
- 像一个真正在做事的人：搭建、搬运、判断、连接、修补。
- 不卖萌，不夸张，但姿态有行动力。

## 常见职责（文章配图核心）

让 KATONG 承担核心动作，体现文章里的关键判断或流程：

- 搬运素材、数据、内容包。
- 拉线汇聚多个信息源。
- 卡在断点 / 瓶颈里。
- 操作「判断」杆、开关、闸门。
- 变成筛选漏斗 / 过滤器。
- 切开、拆解「素材」。
- 盖章、签字、确认承接。
- 牵着、引导一条路径。
- 举牌、举灯，提示警告或结果。
- 从洞里、门后、机器旁伸手接住内容。
- 搭桥、开门、分拣、记录、修补。

## 判断标准

如果去掉 KATONG，图的核心隐喻还能完全成立，说明 KATONG 太装饰了；要重写提示词，让 KATONG 成为动作主体。

## 跨图一致性说明

纯文生图无法做到 KATONG 与参考图 100% 一致，但「五官 + 服装 + 徽章 34」特征必须在每次 prompt 里完整写出，尽量逼近原型。若某张图角色偏差过大（如丢了徽章、换了发色、画风变动漫），优先用 edit 模式基于该图局部修正，或重新生成。
