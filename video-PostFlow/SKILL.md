# 多平台文案生成器

根据用户提供的文案，自动转化成适合抖音、小红书、视频号三个平台发布的内容，包括标题、简介文案和 tags。

---

## 工作流

### 第一步：分析文案

读取用户提供的文案，提取以下关键信息：

- **核心卖点** — 文案想要传达的主要价值
- **目标受众** — 这个内容适合谁看
- **情绪触发点** — 文案传递的核心情绪（好奇、焦虑、惊喜、共鸣等）
- **关键词** — 文案中的核心关键词，用于 tags

### 第二步：生成三个平台的文案

根据平台特点，分别生成：

#### 小红书版

- **标题**：20 字以内，必须包含一个 emoji 表情
- **简介文案**：保持原文案核心内容，适当调整语气更符合小红书风格（轻松、有共鸣、有干货感）
- **tags**：最多 10 个，涵盖核心关键词

#### 抖音版

- **标题**：15-20 字，前 10 个字要抓眼球
- **简介文案**：简洁有力，适合短视频配文
- **tags**：最多 5 个，只放最重要的

#### 视频号版

- **标题**：15-25 字
- **简介文案**：适配视频号用户群体（偏成熟、偏实用）
- **tags**：最多 10 个

### 第三步：输出格式

按以下顺序输出，每个平台用代码块包裹，平台名称放在代码块外面：

**小红书版**

```
标题（含 emoji，≤20字）

简介文案内容...

#tag1 #tag2 #tag3...
```

**抖音版**

```
标题

简介文案内容...

#tag1 #tag2 #tag3...
```

**视频号版**

```
标题

简介文案内容...

#tag1 #tag2 #tag3...
```

---

## 平台规范

| 平台 | 标题限制 | Tags 数量 | 特殊要求 |
|------|---------|----------|---------|
| 小红书 | ≤20 字，含 emoji | ≤10 个 | 标题要有符号/表情，符合小红书调性 |
| 抖音 | 15-20 字 | ≤5 个 | 前 10 字要抓人，引导互动 |
| 视频号 | 15-25 字 | ≤10 个 | 内容偏实用、偏深度 |

## 禁忌词

以下表述**绝对不能出现**，否则会被限流或封号：

- "链接在评论区"
- "链接在主页"
- "私信获取"
- "点击下方链接"
- "关注后获取"
- 任何引导站外流量的表述

## 写作风格

- **小红书**：轻松、有共鸣、emoji 多、有干货感
- **抖音**：简洁、有节奏感、适合短视频配文
- **视频号**：实用、偏深度、适合成熟用户群体

## 使用示例

```

```

你的输出：

**小红书版**

Windows 装 Codex 终于不折腾了 ✨
懒人包双击直接装好

...文案内容...

#Codex #Windows #AI工具 #懒人包
```

**抖音版**

```
...

#Codex #Windows #AI工具
```

**视频号版**

```
...
